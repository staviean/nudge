# Nudge

A neurodivergent-friendly task and to-do integration for Home Assistant, built for people who need their reminders to actually *persist* until a task is done.

Nudge is a fully local task manager with rich scheduling, categories, subtasks, and — its core feature — **nag notifications** that repeat on an interval until you complete or defer a task, with a snooze cap so avoidance becomes a visible signal instead of endless background noise.

## Status

Early development (v0.1.0). Not yet feature-complete.

## Planned Features

- Create, edit, delete, and complete tasks
- Due dates, end dates, and estimated durations
- Categories with custom colors and icons
- Subtasks and descriptions
- Repeating tasks (daily, weekly, monthly, yearly)
- Notifications via mobile push and/or speaker announcements (TTS)
- **Nag notifications** with configurable interval and a snooze cap
- Quiet hours (global, with per-task override)
- "Push to next scheduled time" to defer a task to its next occurrence
- Push to IFTTT
- A custom Lovelace dashboard card for managing everything
